##  Todo App

This project contains a simple TODO application.

###  Features

- Anyone can create a todo  
- Anyone can see their existing todos  
- Anyone can mark a todo as done  

###  Tech Stack

- **Frontend**:  React
- **Backend**: Node.js / Express   
- **Database**: MongoDB
